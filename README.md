# Atari Falcon ("Sparrow") gerber files

Various gerber files for the Atari Falcon 030 have been recovered over the years. They have been publically available for over a decade at this point. 

This repository is an attempt to fix known issues to make them functional as per the original intent.

Version 2 and version 4 of C303006-001 are here. It is unknown how much these differ from the final production pcb files.
